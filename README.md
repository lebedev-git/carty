# Carty — Фабрика учебных колод (MVP)

Автоматизированное создание колод карт по обучению работе с ИИ под любой тип организации.
Архитектура и методология — см. `ПЕРЕСБОРКА_Архитектура_v2.md` и `Создание_карт_АРХИТЕКТУРА.md`.

## Что готово (MVP, Стадия 0)
- Backend (Node + Fastify + SQLite через `node:sqlite`).
- Фикс-каркас «Я»/«МЫ» (6 типов + 6 лучей + паттерны + гаммы) — `server/src/framework.ts`.
- LLM-клиент OpenRouter с надёжным JSON + ретраем — `server/src/llm.ts`.
- **Стадия 0 — Профайлер:** тема → доменное ядро (аудитория, матрица M×C, стоп-листы, тон, боль).
- Автономный веб-редактор: создать колоду, сгенерировать профиль, править матрицу/поля, заморозить.

## Запуск

### 1. Бесплатный ключ OpenRouter
1. https://openrouter.ai → войти → **Keys** → Create Key (бесплатно).
2. Создать файл `server/.env` (по образцу `server/.env.example`):
   ```
   OPENROUTER_API_KEY=sk-or-...
   OPENROUTER_MODEL=google/gemini-2.0-flash-exp:free
   PORT=8787
   ```

### 2. Backend
```bash
cd server
npm install          # уже сделано
npm run dev          # http://localhost:8787
```

### 3. Frontend (в отдельном терминале)
```bash
cd web
npm install          # уже сделано
npm run dev          # http://localhost:5173
```
Открыть http://localhost:5173 — ввести тему (напр. «Школы») → «Создать» →
«Сгенерировать профиль» → проверить/поправить матрицу → «Заморозить».

## API
| Метод | Путь | Назначение |
|---|---|---|
| GET | `/api/framework` | фикс-каркас (типы, лучи, лимиты) |
| GET/POST | `/api/decks` | список / создать |
| GET | `/api/decks/:id` | колода + профиль |
| POST | `/api/decks/:id/profile` | запустить Профайлер (Стадия 0) |
| PUT | `/api/decks/:id/profile` | сохранить правки профиля |
| POST | `/api/decks/:id/profile/freeze` | заморозить профиль |
